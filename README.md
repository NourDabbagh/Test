## HelloWorld Java Projekt

Dieses Projekt ist ein einfaches Java-Programm.

## Beschreibung

Das Programm gibt zwei Sätze in der Konsole aus:
- Hello Welt!
- Wie geht es dir heute?

## Datei 

Die Hauptdatei heißt:

Helloworld.java

## Ausführen des Programms

Zuerst wird die Java-Datei kompiliert

```
javac Helloworld.java
```
Danach wird das Programm gestartet:

```
java HelloWorld
```
